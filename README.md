AI Quote Generator 🤖

A Deep Learning NLP project that generates motivational quotes using an LSTM (Long Short-Term Memory) neural network. The application is deployed using Streamlit and can generate quote-like text from a user-provided starting word or phrase.

🚀 Live Demo

Live App:
https://ai-quote-generator-akshay.streamlit.app/

📂 GitHub Repository
https://github.com/akshay-phulwani/ai-quote-generator


✨ Features
Generate quotes from a starting word or sentence
Built using TensorFlow and LSTM
Interactive Streamlit web interface
Trained on a quote dataset
Deployed online for public access
